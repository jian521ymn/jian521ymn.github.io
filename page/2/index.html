client系列

  • clientwidth：width+左右两个padding值（可视化窗口的高度：）

  • offestparent：父级参照物，当加上position属性（relative，absolute，fiixed）会让该元素所有子孙元素指向当前这个元素

  • offsetwidth/height: 盒子总宽/高度=盒子的左/上–右/下边框的外边框

数据类型检测优缺点总结

###数据类型检测

  • typeof：返回一个字符串，不能检测对象数组正则
  • instanceof：用来检测当前实例是否是某个类，局限性：不能检测自变量出的实例，只要类在当前实例的原型链上都会返回true
  • constructor：可以检测基本数据类型和引用数据类型。局限性：原型可能被扩展，到最后constructor丢失；继承可能导致原型constructor发生改变
  • object.prototype.tostring.call（）：返回值当前函数种this对应的数据类型”[object Object(数据类型)]”；全局下可以直接写tostring.call（）

面向对象原型链与与继承

原型三句大拿（记住这三句，包你一通到底）

  • 每个函数天生自带一个prototype属性，这个属性的只是一个对象
  • prototype的属性值天生自带一个constructor属性，其属性值就是当前原型所属的类
  • 每一个实例都天生自带–proto–属性，这个属性的属性值指向当前实例所属类的原型

    原型链形成

当获取对象的属性名对应的属性值时，先找自己的私有属性；如果没有回默认通过proto向当前类的原型查找。如果原型也不存在，那么会通过原型的proto向继续向上寻找，知道找到object的原型位置，如果也没有，那么就会输出undefined；这样一级一级向上寻找，就会形成原型链。

四大继承模式

  • 原型模式：把类A产生的实例赋值给类B的原型，那么类b的实例就可以使用a的私有属性和共有属性。这种继承方式就是原型继承。
  • 中间类继承：直接更改当前对象的proto指向一个类的原型，那么这个对象就可以使用原型上的方法。
  • call组合继承：继承的目的，让类a产生的实例可以使用类b的私有属性
  • 寄生组合继承：通过call继承了私有属性，结合object.create（）他会创建一个空对象，并让这个空对象指向了传进来的第一个参数，让他指向被继承者的原型

数组的15个方法总结

数组的方法

  • 1.push ;
    • 数组后新增一项或者多项
    • 需要传参
    • 新增以后数组的长度
    • 原数组发生改变
  • 2.pop：
    删除数组的最后一项
    • 不需要传参
    • 删除的那一项
    • 原有数组发生改变
  • 3.unshift：
    • 数组开头新增一项
    • 需要传参
    • 新增以后数组的长度
    • 原数组发生改变
  • 4.shift：
    • 删除数组的第一项
    • 不需要传参
    • 删除的那一项
    • 原数组发生改变
  • 5.splice：
    • 删除数组的某几项
    • splice（m,n）
      • splice（m,n）从索引M开始，删除N个
      • splice（m），从m开始，删除后面所有
      • splice（m，n，z）从m开始，删除n个，用z替代
      • splice（m，0，n）把n项添加到m的前面+
  • 6.slice截取
    • 截取数组中的一项
    • 需要传参
      • slice（m，n）从索引m开始，截取到索引n，不包括n。
      • slice（m）从索引m开始，截取到数组的末尾，m》n，空数组；
      • slice（m，n），如果n<0,则slice（m，length+n），如果m，n都为0（length+m，length+n）
    • 截取的哪几项
    • 原有不发生改变
  • 7.sort排序
    • 排序
    • 需要参数
      • ary.sort(function(a，b){ return a-b; }正序
      • ary.sort(function(a，b){ return b-a; }到序
    • 排序以后的数组
    • 原有数组发生改变
  • 8.reverse：反转
    + 将数组反转
+ 不需要传参
+ 反转之后的数组
+ 原有数组改变
  • 9.concat：拼接
    • 数组的拼接
    • 需要传参
    • 拼接后的数组
    • 原有数组不变
  • 10.join（“”）
    • 将数组成员按照特定的字符串连接到一起
    • 需要传参
    • 返回一个连接字符串
    • 原有数组不发生改变
  • 11.indexof
    • 找到数组成员第一次出现对应的索引，如果不存在，返回值-1，每次只能检测一个
    • 需要传参
    • 返回数组成员第一次出现的索引
    • 原有数组不改变
    • indexof（20,2）寻找200，从第2个索引开始找
  • 12.lastindxof
    • 找到数组成员在数组中最后一次出现对应的索引位置，如果不存在。返回-1，所以常用来判断该项是否存在
    • 需要传参
    • 返回数组成员出现的索引
    • 原数组不改变
  • 13.map：映射，循环
    • ` var len=ary.map(function(item,index){
      console.log(item,index);
