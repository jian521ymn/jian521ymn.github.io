面向对象原型链与与继承

原型三句大拿（记住这三句，包你一通到底）

  • 每个函数天生自带一个prototype属性，这个属性的只是一个对象
  • prototype的属性值天生自带一个constructor属性，其属性值就是当前原型所属的类
  • 每一个实例都天生自带–proto–属性，这个属性的属性值指向当前实例所属类的原型

    原型链形成

当获取对象的属性名对应的属性值时，先找自己的私有属性；如果没有回默认通过proto向当前类的原型查找。如果原型也不存在，那么会通过原型的proto向继续向上寻找，知道找到object的原型位置，如果也没有，那么就会输出undefined；这样一级一级向上寻找，就会形成原型链。

四大继承模式

  • 原型模式：把类A产生的实例赋值给类B的原型，那么类b的实例就可以使用a的私有属性和共有属性。这种继承方式就是原型继承。
  • 中间类继承：直接更改当前对象的proto指向一个类的原型，那么这个对象就可以使用原型上的方法。
  • call组合继承：继承的目的，让类a产生的实例可以使用类b的私有属性
  • 寄生组合继承：通过call继承了私有属性，结合object.create（）他会创建一个空对象，并让这个空对象指向了传进来的第一个参数，让他指向被继承者的原型

字符串的13个方法

字符串的方法

  • 1.touppercase
    把小写字母转成大写；
  • 2.tolowercasr
    把大写字母转成小写字母；
  • 3.chart
    通过索引获取对应字符
  • 4.charcodeat
    通过索引获取对应的字符，根据编码值去输出
  • 5.substr
    字符串的截取（m，n）从m开始截取，截取n个字符
  • 6.subtring
    从索引m开始，截取到索引n，但是不包括索引n
    如果m<n;那么不包括n 如果 m>n 不包括m；如果n是负数，直接截取到开头
  • 7.slice
    如果是负数，需要将其转成正数。用法差不多和substring。他可以截取负数
    length+负数去转变
  • 8.indexof
    返回字符在字符串中第一次出现的位置：入伙不存在，返回-1
  • 9.lastindexof
    返回字符在字符串中最后一次出现的位置：入伙不存在，返回-1
  • 10.concat
    字符串的拼接
  • 11.split
    按照特定的字符把字符串分割成数组中的每一项，返回一个数组，如果里面是,就改成数组
  • 12.replace
    替换；replace（老字符串，新字符串）
  • 13.trim
    去左右空格（right）（left）

数组的15个方法总结

数组的方法

  • 1.push ;
    • 数组后新增一项或者多项
    • 需要传参
    • 新增以后数组的长度
    • 原数组发生改变
  • 2.pop：
    删除数组的最后一项
    • 不需要传参
    • 删除的那一项
    • 原有数组发生改变
  • 3.unshift：
    • 数组开头新增一项
    • 需要传参
    • 新增以后数组的长度
    • 原数组发生改变
  • 4.shift：
    • 删除数组的第一项
    • 不需要传参
    • 删除的那一项
    • 原数组发生改变
  • 5.splice：
    • 删除数组的某几项
    • splice（m,n）
      • splice（m,n）从索引M开始，删除N个
      • splice（m），从m开始，删除后面所有
      • splice（m，n，z）从m开始，删除n个，用z替代
      • splice（m，0，n）把n项添加到m的前面+
  • 6.slice截取
    • 截取数组中的一项
    • 需要传参
      • slice（m，n）从索引m开始，截取到索引n，不包括n。
      • slice（m）从索引m开始，截取到数组的末尾，m》n，空数组；
      • slice（m，n），如果n<0,则slice（m，length+n），如果m，n都为0（length+m，length+n）
    • 截取的哪几项
    • 原有不发生改变
  • 7.sort排序
    • 排序
    • 需要参数
      • ary.sort(function(a，b){ return a-b; }正序
      • ary.sort(function(a，b){ return b-a; }到序
    • 排序以后的数组
    • 原有数组发生改变
  • 8.reverse：反转
    + 将数组反转
+ 不需要传参
+ 反转之后的数组
+ 原有数组改变
  • 9.concat：拼接
    • 数组的拼接
    • 需要传参
    • 拼接后的数组
    • 原有数组不变
  • 10.join（“”）
    • 将数组成员按照特定的字符串连接到一起
    • 需要传参
    • 返回一个连接字符串
    • 原有数组不发生改变
  • 11.indexof
    • 找到数组成员第一次出现对应的索引，如果不存在，返回值-1，每次只能检测一个
    • 需要传参
    • 返回数组成员第一次出现的索引
    • 原有数组不改变
    • indexof（20,2）寻找200，从第2个索引开始找
  • 12.lastindxof
    • 找到数组成员在数组中最后一次出现对应的索引位置，如果不存在。返回-1，所以常用来判断该项是否存在
    • 需要传参
    • 返回数组成员出现的索引
    • 原数组不改变
  • 13.map：映射，循环
    • ` var len=ary.map(function(item,index){
      console.log(item,index);
/* item函数的每一项，index是索引 */ `
    • 输出每一项
    • 需要传参
    • 返回值是一个映射新数组
    • 原数组不变
  • 14.forEach循环
    • 没有return，返回值是undefined
    • var len=ary.foreach(function(item,index){
    • 需要传参
  • 15.toString：转字符串
    • 转字符串
  • 不需要传参
    • 返回值是字符串
    • 原数组改变
