数组的15个方法总结

数组的方法

  • 1.push ;
    • 数组后新增一项或者多项
    • 需要传参
    • 新增以后数组的长度
    • 原数组发生改变
  • 2.pop：
    删除数组的最后一项
    • 不需要传参
    • 删除的那一项
    • 原有数组发生改变
  • 3.unshift：
    • 数组开头新增一项
    • 需要传参
    • 新增以后数组的长度
    • 原数组发生改变
  • 4.shift：
    • 删除数组的第一项
    • 不需要传参
    • 删除的那一项
    • 原数组发生改变
  • 5.splice：
    • 删除数组的某几项
    • splice（m,n）
      • splice（m,n）从索引M开始，删除N个
      • splice（m），从m开始，删除后面所有
      • splice（m，n，z）从m开始，删除n个，用z替代
      • splice（m，0，n）把n项添加到m的前面+
  • 6.slice截取
    • 截取数组中的一项
    • 需要传参
      • slice（m，n）从索引m开始，截取到索引n，不包括n。
      • slice（m）从索引m开始，截取到数组的末尾，m》n，空数组；
      • slice（m，n），如果n<0,则slice（m，length+n），如果m，n都为0（length+m，length+n）
    • 截取的哪几项
    • 原有不发生改变
  • 7.sort排序
    • 排序
    • 需要参数
      • ary.sort(function(a，b){ return a-b; }正序
      • ary.sort(function(a，b){ return b-a; }到序
    • 排序以后的数组
    • 原有数组发生改变
  • 8.reverse：反转
    + 将数组反转
+ 不需要传参
+ 反转之后的数组
+ 原有数组改变
  • 9.concat：拼接
    • 数组的拼接
    • 需要传参
    • 拼接后的数组
    • 原有数组不变
  • 10.join（“”）
    • 将数组成员按照特定的字符串连接到一起
    • 需要传参
    • 返回一个连接字符串
    • 原有数组不发生改变
  • 11.indexof
    • 找到数组成员第一次出现对应的索引，如果不存在，返回值-1，每次只能检测一个
    • 需要传参
    • 返回数组成员第一次出现的索引
    • 原有数组不改变
    • indexof（20,2）寻找200，从第2个索引开始找
  • 12.lastindxof
    • 找到数组成员在数组中最后一次出现对应的索引位置，如果不存在。返回-1，所以常用来判断该项是否存在
    • 需要传参
    • 返回数组成员出现的索引
    • 原数组不改变
  • 13.map：映射，循环
    • ` var len=ary.map(function(item,index){
      console.log(item,index);
/* item函数的每一项，index是索引 */ `
    • 输出每一项
    • 需要传参
    • 返回值是一个映射新数组
    • 原数组不变
  • 14.forEach循环
    • 没有return，返回值是undefined
    • var len=ary.foreach(function(item,index){
    • 需要传参
  • 15.toString：转字符串
    • 转字符串
  • 不需要传参
    • 返回值是字符串
    • 原数组改变
